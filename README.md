# InventoryPro

Sistema em Python com interface Flet para controle de equipamentos e patrimônios com SQLite

## Arquivos principais
- main.py
- ui.py
- data.py
- inventory.db

## Tecnologias usadas
- Python
- Flet
- SQLite
- Pandas
- FPDF
- python-docx

## Como executar
1. Instale o Python 3.10 ou superior
2. Instale as dependências:
   pip install -r requirements.txt
3. Execute:
   python main.py

## Como usar o programa
1-Faça o login com qualquer senha(para fins didáticos, deixei qualquer senha como válida);

2-Vá até a aba "suporte" do programa e siga as instruções que constam lá.

## Observação
O sistema usa o arquivo `inventory.db` como banco de dados SQLite.
